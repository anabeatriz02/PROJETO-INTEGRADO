FPOO

JAVA

Java é uma linguagem de programação e plataforma computacional rápido, confiável e seguro. Tem diversos sites e aplicações que não funcionarão, a menos que você tenha o Java instalado, o Java está em todos os lugares!!!

ECLIPSE

O Eclipse é uma IDE para desenvolver Java, mas é possível desenvolver outras linguagens de programação também. Ele foi feito em Java e segue o modelo open source de desenvolvimento de software

LIMA

HTML

HTML é uma linguagem de marcação de hipertexto, é utilizada como marcação para desenvolver páginas e documentos eletrônicos para a internet , ou seja, garante a formatação ideal para sites.

CSS

O CSS é uma “folha de estilo” composta por camadas e utilizada para definir a apresentação, ou melhor, a aparência em páginas da internet que adotam para o desenvolvimento das linguagens de marcação

HARE

HARDWARE

Hardware de rede: são dispositivos, equipamentos que facilitam e dão suporte ao uso de uma rede de computadores, ou melhor, são meios físicos necessários para a comunicação entre os componentes participantes de uma rede

REDE

As redes de computadores podem ser definidas como um conjunto de equipamentos, que além de compartilhar dos mesmos recursos, também podem trocar informações entre si.

SOP

WINDOWS

O Windows é um sistema operacional criado pela Microsoft que facilita o acesso do usuário ao computador, com interfaces mais eficientes, atrativas e fáceis de utilizar, alem de integrar softwares mais rápidos e eficazes

LINUX

O Linux é um sistema operacional criado pelo Linus Torvalds que possibilita a execução de programas em um computador e outros dispositivos. O Linux pode ser livremente modificado e distribuído...
